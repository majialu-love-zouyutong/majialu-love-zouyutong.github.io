CPU

主要有 AMD 和 Intel

主板

主板必须与 CPU 相匹配

内存

显卡

PCIe 插槽

高端显卡有额外供电接口

硬盘

机械硬盘
通常使用 SATA 接口,速度慢,价格低

  • 3.5 寸
  • 2.5 寸

固态硬盘

  • 2.5 寸固态: 与机械硬盘接口一样
  • PCIe 固态
  • M.2 固态

散热

有风冷和一体式水冷

电源

交流变直流

  • 非模组电源: 所有电源线从电源内部出来,不可拆卸
  • 半模组电源: 必须线缆直出,其他可拆卸
  • 全模组电源: 所有电缆都可拆卸

接口

  • 24Pin 主板供电: 给主板供电,有一个脚是空的
  • 4+4Pin CPU 供电: 给 CPU 供电
  • 6+2Pin PCIe 供电: 给显卡和 PCIe 设备供电
  • SATA 供电: 扁 L 供电,给 SATA 设备供电
  • 大 4Pin 供电: 大 4D 供电,外围设备供电,灯条,集线器等.可转换为其他类型供电,也叫万能供电.
